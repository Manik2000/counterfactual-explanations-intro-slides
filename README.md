# Counterfactual explanations introduction

Slides for my lecture on counterfactual explanations for students research group Gauss.

Slidev is used for creating slides with some addons and custom components. To learn more about Slidev, please visit https://sli.dev/.

# Quick setup

Provided you have `bun`, run

```bash
bun install
```

to get the required dependencies.

Afterwards, `bun run dev` will open the presentation on `localhost:3030`. To get `dist`, please run `bun run build`.
